Promise 是 JavaScript 中处理异步操作的一种机制，它能够让异步代码看起来更像同步代码，从而提高代码的可读性和可维护性。Promise 的出现解决了传统回调函数嵌套过深（即“回调地狱”）的问题，是现代 JavaScript 开发中不可或缺的工具。 Promise 通过 new Promise() 构造函数创建。构造函数接受一个函数（称为 executor），这个函数有两个参数：resolve 和 reject。 resolve(value)：将 Promise 的状态从 Pending 变为 Fulfilled，并传递成功的结果。 reject(error)：将 Promise 的状态从 Pending 变为 Rejected，并传递失败的原因。 Promise 提供了两个主要方法来处理异步操作的结果： .then(onFulfilled, onRejected)：当 Promise 变为 Fulfilled 时，调用 onFulfilled 函数；当变为 Rejected 时，调用 onRejected 函数。 .catch(onRejected)：专门用于捕获 Rejected 状态的错误，相当于 .then(null, onRejected)。
